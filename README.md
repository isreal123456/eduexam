# EduExam - Django Exam System

EduExam is a web-based examination system built with Django. It allows students to register, take exams, and view their results. Staff can manage subjects and questions.

## ✨ Features

- Custom user model with student/staff roles
- Exam creation and management
- One-time exam submission with timer
- Auto-submission on timeout
- Email-based password reset
- Result tracking and report view
- Responsive UI using Bootstrap
