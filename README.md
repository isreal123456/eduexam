# 📝 Student Exam Portal

A web-based examination system built with **Django**. This platform allows educational institutions to manage subjects and questions while enabling students to take timed online exams and view their results.

---

## 🚀 Features

### 👨‍🏫 Staff
- Add, edit, and delete subjects
- Add, edit, and delete multiple-choice questions
- View all students' results and performance

### 👨‍🎓 Students
- Browse available subjects
- Take timed exams (one attempt per subject)
- View their own results and scores

---

## 🔧 Technologies Used

- **Python 3**
- **Django 4**
- **SQLite3** (default database)
- **Bootstrap 5** for responsive UI
- **HTML/CSS/JS** for frontend design
